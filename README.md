# Punchline WebApp

This is a simple flask app that returns a punchline an id


## How To run
1) Install requirements
   1) ``` 
      pip install -r requirements.txt
      ```
2) Run application
   1) ```
      python <PATH_TO_SCRIPT>
      ```
   2) Consume your punchline
      In your browser go to http://127.0.0.1
